## ✨ Особенности
- 📅 Удобный календарь с навигацией по дням
- 📝 Текстовый редактор с базовым форматированием
- 🌍 Поддержка двух языков: Русский / English
- 🎨 Настройка шрифта, интервала и гарнитуры
- 🔄 Скролл даты колёсиком мыши (дни/месяцы/годы)
- 📊 Экспорт записей (день / все)
- 💾 Автосохранение в localStorage
- 📱 Адаптивный дизайн

## 🛠 Использование
1. Откройте [веб-приложение](https://somenmi.github.io/widty/)
2. Выберите дату в календаре или используйте навигацию ← →
3. Пишите заметки с форматированием
4. Переключайте даты скроллом мыши (Shift: ± Год / Alt: ± Месяц / Только скролл: ± День)
5. Настройте внешний вид в меню ⚙️
6. Экспортируйте записи в TXT

## ⌨️ Горячие клавиши
- `Только Скролл` - дни · `Alt + Скролл` - месяцы · `Shift + Скролл` - годы 
- `Esc` - закрыть календарь

## 💾 Хранение данных
localStorage:
- Заметки: `whatIDidToday_YYYY-MM-DD`
- Настройки: `whatIDidToday_settings` (язык, шрифт, интервал)
